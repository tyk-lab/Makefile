使用说明：

只需要在src目录下，添加源文件，就会自动识别到添加的文件，添加 .o 依赖到最终目标中

该版本中，添加了自动头文件依赖，只要头文件的内容有改变，就会重新刷新工程

由于makefile的特性，只会编译改动过的相关文件

使用方法：
1. 在 src 或 inc 中，添加 或 修改自己的源文件 或 头文件
-> 可以在makefile中，修改最终的可执行文件名字，修改编译相关选项，修改编译器类型

2. 直接make
-> make debug 查看变量是否正确

注：
1. 最终的可执行程序，没有放到 bin目录中，在makefile目录中
2. 无论所编译的 .o 文件有没有用到，都会被加进最后的可执行文件中
3. 头文件只是添加，不会编译工程
